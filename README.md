# **Note**
- When renaming weapons, as long as the required name (e.g. Buster Sword) is included in the whole name, the change works. (e.g. VoidBuster Sword, My Needle of Fate, Needle of Dreams and stuff)

- HUD changes wil be carried across all available versions while weapon texture changes are along 1.17 - 1.21.1 with CIT Resewn or all versions with Optifine

- Although the resolution tag says 128x, it is highly inconsistent as I optimise the textures relative to model size to give them more detail

- Emissive textures do not work on later versions

- I recommend using Better Combat along with this it looks so good

- I also recommend using RP Renames for more efficient renaming (Doesnt work on later versions when use CIT Resewn Continuation)

- Resource pack updates would be more delayed now due to busy schedule but will have (hopefully) more content in every release. Sorry for the inconvenience caused.

# **For NeoForge and Forge users**

**NeoForge:**
- Install [Sinytra Connector](https://modrinth.com/mod/connector)
- Install [Forgified Fabric API](https://modrinth.com/mod/forgified-fabric-api)
- Install [CITResewnNeoPatcher](https://modrinth.com/mod/cit-resewn-neopatcher)
- Install [CIT Resewn](https://modrinth.com/mod/cit-resewn) / [CIT Resewn Continuation](https://modrinth.com/mod/cit-resewn-continuation)
- You should be able to use this and other CIT packs once these 4 mods has been installed

**Forge:**
- Sorry but u can't use this pack for Forge. You can consider switching to NeoForge, Fabric, or Quilt
